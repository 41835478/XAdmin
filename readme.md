## Laravel5-Backend

基于Laravel5.1(LTS) 开发

## 功能列表

### 用户登录
* 参考前端登录做后台登录
### 用户管理
* 用户列表：主要用来查看用户信息
### 权限管理
* ACL (Access Control Layer) using Laravel Middleware
### 新闻管理
* 分类管理
* 列表管理
* 创建新闻
* 标签管理
### 单页管理
* 分类管理
* 内容管理
### 相册管理
### 活动管理
### 评论管理
### ...
